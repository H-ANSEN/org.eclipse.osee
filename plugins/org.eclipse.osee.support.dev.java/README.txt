This package contains Eclipse JDT / Java tips and tricks for developers.

All training items and examples are notated by // JavaTip so you can access via the Tasks view in Eclipse.

1) Windows -> Preferences -> Java -> Compiler -> Task Tags -> Add "JavaTip"
2) Open Tasks view -> Right white triangle pulldown -> Configure Contents
2a) New "Java Tips" - set contains to "JavaTip" - Ok
2b) Select "JavaTip" - Push Default button
2c) In Configurations, Un-select TODOs and select Java Tips
2d) Select Ok
